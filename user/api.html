API 관리

※ 거래소에서 발급받은 API Key는 꼭 복사 후 붙여넣기 방법으로 등록해 주십시오.
※ 아비봇에서 지원하는 모든 거래소는 1개의 API만 등록하여 사용 가능합니다.
※ 업비트 API 발급 시 안내되어 있는 IP를 사용해야지만 아비봇을 정상적으로 이용할 수 있습니다.
※ 업비트는 API Key 검증 기능을 제공하지 않습니다.
※ 운영중인 봇이 있을 경우 API를 삭제할 수 없습니다.
※ 회원님의 API Key는 2중 암호와 망분리된 시스템에서 관리됩니다.
