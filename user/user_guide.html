이용 안내

안녕하세요.
가상자산 멀티 트레이딩 플랫폼 "아비봇" 입니다.

코인 투자가 처음인 분부터 트레이딩 고수까지, 모두에게 맞춤화 된 트레이딩 봇을 제공하는 아비봇의 사용 방법을 지금부터 알아보겠습니다.

STEP 1. 회원가입

① 사이트 우측 상단에 회원가입 버튼을 클릭해주세요.
② 카카오 또는 네이버 아이디로 로그인해주세요.

STEP 2. API 등록

① 이용하는 거래소에 로그인하여 Key값을 발급받은 후 복사해주세요.
② 각 거래소별 발급하는 방법은 [거래소 이름] 클릭 후 탭메뉴 하단 우측의 [API 발급 가이드]를 참고해주세요.

※ 거래소마다 API Key의 명칭이 다르니 잘 읽고 입력해주세요.
※ Secret Key는 나중에 ** 별표 처리되어 가려지므로, 생성됐을 때 복사하여 별도 메모해주세요.
※ 업비트의 경우 IP도 추가로 입력해주세요.

STEP 3. 봇 설정

① 교차거래 봇

  1. 일정한 가격범위 안에서 자동으로 저가에서 매수하고 고가에서 매도하여 시세차익에 따른 수익을 얻는 봇입니다.
  2. 횡보장이나 변동이 심한 장일 때 적합한 전략입니다.
  3. 각 설정 항목의 i표시를 누르면 자세한 설명이 있습니다.

② 재정거래 봇

  1. 거래소 간의 시세 차이를 이용하여 코인의 수량을 유지하고 원화(KRW) 를 늘리는 전략입니다.
  2. 장기적으로 봇을 운영했을 때 수익이 안정적인 전략입니다.
  3. 각 설정 항목의 i표시를 누르면 자세한 설명이 있습니다.

기타 문의사항은 카카오채널로 문의 주세요.
