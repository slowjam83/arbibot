알림

※ 알림 보관 기간은 1개월로 만료된 데이터는 오래된 순부터 삭제됩니다.

공통

  1. 사용자 종료 : 사용자가 운영중인 봇을 종료한 경우
  2. 포인트 부족 종료 : 포인트 부족으로 정지 후 72시간 이후까지 포인트 충전이 안 된 경우

교차거래

  1. 밸런스 주문 오류 종료 : 밸런스 주문 오류가 발생 했을 경우
  2. 주문 취소 종료 : 사용자가 거래소에서 주문을 취소한 경우
  3. 로스컷 종료 : 사용자가 설정한 로스컷보다 코인 현재 가격이 더 낮은 경우
  4. 주문 오류 종료 : 주문 오류가 발생한 경우
  5. 수수료 초과 종료 : 1회 페어 거래당 수수료가 수익금보다 더 큰 경우

재정거래

  1. 사용자 정지 : 사용자가 재정거래 봇을 정지한 경우
  2. 포인트 부족 정지 : 보유 포인트가 2,000 포인트 이하인 경우
  3. 미거래 종료 : 봇의 마지막 거래 후 120시간 이후까지 거래가 없는 경우
