아비봇 이용안내 확인

아비봇은 가상자산의 시세를 올리거나 내리는 기능을 하지 않습니다.

아비봇은 가상자산 트레이딩을 편리하게 이용할 수 있는 도구만 제공하며, 투자권유 또는 유치를 목적으로 하지 않습니다.

가상자산 급격한 시세 변동 상황 또는 가상자산 거래소의 API 서비스 불안정으로 인한 투자손실에 책임지지 않으므로 서비스 이용에 유의하시기 바랍니다.

아비봇을 이용하는 회원님은 모든 이용 약관을 읽고 이해하였고, 동의하셔야만 이용이 가능합니다.

슬리피지 안전거래 안내

슬리피지 안전거래 기능은,

슬리피지(미체결 주문)가 발생하면 셋팅된 대기 시간까지 대기 후 앞선 발생된 슬리피지(미체결 주문)를 취소하고, 즉시 체결될 수 있는 호가에 슬리피지 발생 수량의 주문을 넣는 기능입니다.

슬리피지 안전거래 활용

코인의 시세 변동성과 거래량에 따라 슬리피지가 발생하여도 짧은 시간 내에 주문이 체결되는 경우도 있습니다. 다만 슬리피지가 발생한 시세로 수 일 또는 셋팅한 봇을 삭제할 때 까지 돌아오지 않을 경우 손실이 발생 할 수 있어 운영 전략에 맞춰 기능을 활용하시는 것이 좋습니다.

빗썸 API 발급 가이드

1. 로그인 후 상단 우측의 ≡ 클릭

2. 계정관리 > API관리 클릭

3. API 활성화 항목 설정

① 활성화 항목을 체크합니다.

  1. 회원지갑정보
  2. 주문내역
  3. 매수주문
  4. 매도주문
  5. 거래취소
  6. 회원거래내역

② 선택금지 항목은 절대 체크하지 마세요.

  1. KRW 출금
  2. 가상자산 출금

4. 보안 인증

① 보안비밀번호 4자리를 입력해주세요.
② ARS 인증요청을 클릭하여 인증해주세요.
③ ARS 인증요청을 누르면 승인번호가 나옵니다.
④ 휴대폰으로 걸려 온 전화를 받아 승인번호를 입력해주세요.
⑤ 통화가 종료되면 <전화승인완료>를 클릭합니다.

5. API KEY 생성

① ARS 인증 완료 후 API KEY 생성을 클릭합니다.
빗썸 API KEY 생성안내 팝업창이 나오면 ② <API 이용 및 주의사항 확인하였습니다.> 체크
③ 확인 버튼을 누릅니다.
API KEY 생성 완료 안내창 ④ 확인 클릭

6. 사용 중 API 리스트

① 새롭게 생성된 API 리스트를 확인할 수 있습니다.
② <활성화> 버튼을 눌러주세요. (고객님의 연락처로 문자가 발송됩니다.)

7. API Key 발급 완료

(새로 발급받은 API 활성화 버튼을 클릭 후) ① 수신 받은 문자의 링크를 클릭해주세요.
② 링크페이지가 열리고 API Key 활성화 완료됩니다.
③ API 활성화 완료 새롭게 생성된 Secret Key는 꼭 복사하여 별도 메모하여 주세요. 첫 생성 후 다시 확인할 수 없습니다. 발급된 Connect Key와 Secret Key를 아비봇 API에 등록해주세요.
